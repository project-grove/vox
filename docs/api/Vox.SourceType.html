SourceType

Enum

Namespace: Vox

Declared in: Vox

Indicates whether a source is static (has one SoundBuffer attached) or streaming (has a queue).false. If it's undetermined, the target status will be changed automatically on modification.

Enumeration

Underlying type: int

Values:

  • Undetermined = AL_UNDETERMINED
  • Static = AL_STATIC
  • Streaming = AL_STREAMING
