# SuspendProcessing()

**Method**

**Namespace:** [Vox](Vox.md)

**Declared in:** [Vox.OutputDevice](Vox.OutputDevice.md)

Suspends the output device processing (if wasn't suspended). That
means the sound playback offsets will NOT be incremented.
